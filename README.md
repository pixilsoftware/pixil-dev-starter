# Pixil Dev Starter

A small open-source starter kit of templates and snippets we use across Pixil projects.

## What’s inside
- README templates
- .gitignore / .editorconfig examples
- Contribution guidelines
- Basic project structure suggestions

## Who is this for?
Developers building small web tools, dashboards, or internal utilities who want a clean starting point.
